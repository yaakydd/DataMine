Here's the full breakdown of every decision.
recommend_transform() — why the decision tree is structured this way. The first check is always has_negatives because that immediately rules out log, log1p, sqrt, Box-Cox, and reciprocal in one go — Yeo-Johnson is the only safe choice. Then it branches on skewness direction. For right skew, the choice between log and log1p depends purely on whether zeros exist — log(0) is undefined, so log1p (which computes log(1+x)) is the automatic fallback. Moderate right skew gets sqrt because it is gentler and less likely to overcorrect. Left skew goes straight to Yeo-Johnson because log and sqrt only fix right skew — applying them to left-skewed data makes things worse.
series.dropna() before all calculations. Same reasoning as Task 4 — NaN propagates silently through skewness calculations and would return nan as the score. We drop them from the calculation only, not from the DataFrame.
scipy Box-Cox and Yeo-Johnson return a tuple. stats.boxcox(series) returns (transformed_array, optimal_lambda). We unpack both — the transformed values go back into the DataFrame, and the lambda is included in the response so the user can see what transformation power was chosen. A lambda of 0 means log, 0.5 means sqrt, 1 means no transformation was needed.
The NaN-preserving assignment for Box-Cox and Yeo-Johnson. scipy operates on the array of non-null values only, so it returns a shorter array. We can't directly assign it back to df[col] because the lengths don't match. The pattern result[result.notna()] = transformed creates a copy of the column, finds all positions that are not NaN, and fills only those positions with the transformed values — leaving NaN positions untouched.
Before/after skewness in the response. explain_transform_result() compares skew_before and skew_after and tells the user whether the transformation actually improved things. If it didn't, the message says so and suggests trying a different method. This is the most useful feedback you can give — without it the user has no way to know if their choice worked.
